        Известные проблемы: 
        * Ретвиты обрезаются.

        Берём поля:
        * retweet - \lstinline{parsed_tweet['retweeted'] or text.startswith('RT @')}
        * text.
        * created\_at - время по гринвичу.
        * timestamp\_ms - timestamp по местному времени (в нашем случае +0300).
        * lang.
